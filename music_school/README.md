# Music School CRM

Этот проект содержит фронтенд на HTML/CSS/JS и простой Node.js бэкенд для безопасной работы с Supabase.

## Защитить ключи

1. Скопируй `.env.example` в `.env`.
2. Заполни `SUPABASE_URL` и `SUPABASE_SERVICE_ROLE_KEY`.

## Запуск

1. Установи зависимости:
   ```bash
   npm install
   ```
2. Запусти сервер:
   ```bash
   npm start
   ```
3. Открой в браузере:
   - `http://localhost:3000/index.html`
   - `http://localhost:3000/schedule.html`
